第 0x0004 章 未定义

    很快，在春节后的第七天，我们就返校了。
    上学期推迟的期末考试会在1月40日到1月41日进行，但考虑到我们上网课的情况，从空间向量重学一遍也不是不可思议的操作。毫无疑问的，在十天内复习一遍不具有较大的可能性，这不是好的。
    周二下午信息课我边在ArchLinux的KDE Plasma的Konsole敲指令边看右边同学观赏Blue Archive视频时，突然感到意识进程的一部分被中断，有超出数据处理能力的数据输入进一个没有定义的接口，在大脑的数据流里四处游荡。头疼的似乎要长出额外的大脑一样，我非条件反射般的用手按住头。但是，在屏幕上的时钟走了几秒之后，头疼感消失了。旁边的同学好像察觉到了什么，但是看到我打开了Minecraft，他继续观看视频。我试图思考刚才发生了什么，但是似乎就像只存在这种感觉一样，没有额外的数据被保存进大脑。日志里记录了一个线性地址，但是这个地址没有定义，在我试图访问这个地址时，没有报错，但是到超时也没有任何返回。
    在周六下午，我在达成一周内速通椭圆双曲线抛物线的成就之后，胜利归来般的坐上了回家的公交。放假两天，刚好可以过元宵节。
    下车，步行向家移动。
    走着走着，我突然感到很头晕，就像有什么要出来了一样，脚步也停了下来，人晃了几下，但终归是没有摔倒。这似乎不是什么大问题，我继续移动，在路上进入菜鸟驿站获取了快递，持续移动，直到我推开了我家的门，坐在分别已久的沙发上。
    干完晚饭，刚修了几个特性就感到瞌睡，没有办法，我只得怀着期待着元宵节的心情入睡。
    元宵节晚上，我前往新区并观看了烟花表演。
    无数的火光从地上飞到空中，绽放绚丽的色彩。有一个烟花，飞了十几秒，在天空中展开了一层绿色的包络线，又迅速的展开了一层红色的包络线，包络线末端的火光又在变得较为暗淡时突然炸亮，大量的白色的包络线在空中描绘出繁华的景象。
    烟花易逝，人情长存。当初陪我看烟花的人已经永远离开了，或许以后也会有曾陪我看烟花的人离开，最终我也一定会离开所有陪我看烟花的人。世界多次改变，烟花在这片土地上存在了上千年，甚至可以预期在将来很长一段时间内烟花也将陪伴着我们。人情长存，而人易逝；烟花易逝，而烟花长存。
    回到家，我开始听音乐。
    听着听着，熟悉的旋律传入耳朵，看了一眼播放器，Vocaloid原创曲。明明每次都说着不术了要离开了，可是每次听着​熟悉的旋​律总是鼻头一酸，又默默爬了回来，只要他们还一直唱下去，我就会一直听下去​。
    仍然记得，在2018年夏天，我站在开放式阳台边，身上被蚊子咬了一个又一个包，只为了蹭远方的WIFI，听一曲Vocaloid原创曲。当时家里还没有装宽带，我也还不知道如何Kali得到WIFI密码。入的还是晚了一点，主要还是考古。虽然入的还是晚了，但我最开始总是在听术曲，多少记忆里都包含着术曲的曲调。这对我来说不只是音乐，也不只是人设，更不只是故事，这是我记忆的一部分，也是我生命的一部分。对人类来说，记忆是一个人重要的部分，没有记忆，一切都变成了默认值。对于依靠记忆来进行决策的人脑而言，记忆是必须的，不可缺少的一部分，也是我们生命的重要组成部分。你可以改变对过去的看法，但完全忘记是不可能的，毕竟某些长期的第三级记忆已经成为物理结构，作为大脑的一部分，几乎不能在没有损害的状况下移除。
    第二天起来，看了看本上记的内容，我突发奇想要让我的服务器变成机娘。
    我依照经典的方法在眼前调出了一个像Visual Studio的工作环境，然后打开API调用规范看了半天。我发现我必须为服务器创建一个灵魂才能使服务器具有真正的独立思考的能力，但是我不知道灵魂固件区域里有什么数据，这文档里只写了一些参数的标准，而默认值和区域开头很长一段都完全没写。于是我试图以只读方法镜像一遍自己灵魂的固件区域，这样我就可以分析灵魂的固件区域有什么了。
    我刚逝图用灵魂镜像工具选择了我自己灵魂的固件区域，它就弹出了一个错误提示框：Fatal Error\nFail to read a part of the data or all of the data.\nIf this is the first time you see this error, retry.\nIf you have seen this for more than twice, try execute the initialize function of target data part.\0。
    我斟酌半天，最终在大脑里创建了一个指向灵魂固件区域开始地址的线程，但是眼前突然出现了一个像用户账户控制的提示框，要求对授权操作宇宙底层数据的权限。我还没看完提示框内容，手一滑，碰到了确定按钮。
    眼前的世界变得光怪陆离且无法描述起来，似乎有什么正在修改我大脑中的数据，我艰难的思考着，但还没来得及补救一下，意识就被中断了。
